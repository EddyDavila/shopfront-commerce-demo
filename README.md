# Northline Goods — E-commerce Demo

A responsive storefront concept with category filtering, cart management, totals, and a simulated checkout flow. It demonstrates how BitterSoft can pair polished commerce experiences with AI-ready sales insights and performance reporting.

Created by **Eddy Davila** of [BitterSoft](https://eddydavila.github.io/).

This is a portfolio demonstration. No real orders or payments are processed.
